# A Quick Test Project in Swift on Linux

## Preamble:

A basic env can be set up using [my swift on ubuntu/linux setup](https://github.com/rudenoise/swiftOnUbuntu).

## Run:

Clone this repo, step inside and...

```
swift build
.build/debug/hello rudenoise
swift test
```
